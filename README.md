# Tableau Projects Portfolio

This repository contains Tableau dashboard projects developed for data analysis, business intelligence, and data visualization.

## Projects

### Tableau Project 1 – RCDC Research Funding Analysis Dashboard

- Analyzed research funding trends from 2008–2022.
- Identified funding growth patterns across research and disease areas.
- Evaluated funding concentration and distribution using interactive visualizations.
- Developed a Tableau dashboard featuring line charts, waterfall charts, heat maps, funnel charts, and comparative analysis views.
- Generated insights into long-term funding trends and research investment priorities.
